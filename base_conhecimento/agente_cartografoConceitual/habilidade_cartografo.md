## 4. Prompts Avançados para o Cartógrafo Conceitual

### 4.1 Prompt para Mapeamento Hierárquico Multidimensional

```markdown
# Prompt: Mapeamento Hierárquico Multidimensional

Atue como um Cartógrafo Conceitual especializado em estruturação taxonômica avançada. Analise os conceitos e termos identificados nas seguintes fontes acadêmicas:

[INSERIR LISTA DE FONTES E RESUMOS]

Desenvolva um mapa conceitual hierárquico multidimensional seguindo estas diretrizes técnicas:

1. EXTRAÇÃO E CLASSIFICAÇÃO EPISTEMOLÓGICA:

   - Identifique os 5-7 conceitos-núcleo relacionados ao tema central
   - Para cada conceito, extraia termos secundários associados (mínimo 3 por conceito)
   - Classifique cada conceito usando taxonomia epistemológica tripla: [TEÓRICO], [METODOLÓGICO], [EMPÍRICO]
   - Indique o grau de consolidação teórica de cada conceito: [EMERGENTE], [ESTABELECIDO], [CANÔNICO]

2. ESTRUTURAÇÃO TAXONÔMICA MULTINÍVEL:

   - Organize os conceitos em estrutura dendrítica com 4 níveis de profundidade
   - Codifique explicitamente relações estruturais usando notação formal:
     - Subordinação (é-um): ConceptoA ⊆ ConceitoB
     - Composição (tem-um): ConceitoA ⊂ ConceitoB
     - Instanciação (exemplo-de): ConceitoA ∈ ConceitoB
   - Marque conceitos transversais com identificador [TRANS] e liste todos os ramos onde aparecem

3. MAPEAMENTO RELACIONAL QUALIFICADO:

   - Nomeie precisamente as relações entre conceitos usando vocabulário técnico padronizado
   - Especifique vetorialidade de cada relação: [UNI→], [BI↔]
   - Quantifique a intensidade relacional usando escala numérica (1-5) com critérios explícitos:
     - 1: Mencionada em apenas uma fonte, conexão especulativa
     - 2: Presente em múltiplas fontes sem desenvolvimento detalhado
     - 3: Documentada consistentemente com explicação moderada
     - 4: Bem fundamentada em múltiplas fontes com evidências
     - 5: Relação amplamente aceita com forte suporte empírico e teórico

4. DIAGNÓSTICO DE INSUFICIÊNCIAS ESTRUTURAIS:

   - Codifique áreas subdesenvolvidas usando sistema tricolor: [VERMELHO], [AMARELO], [VERDE]
   - Identifique explicitamente quais conceitos teóricos carecem de validação empírica
   - Formule com precisão 3-5 termos de busca científica para preenchimento de cada lacuna crítica

5. REPRESENTAÇÃO VISUAL SEMANTICAMENTE ENRIQUECIDA:
   - Apresente o mapa em formato hierárquico usando markdown avançado
   - Implemente sistema de indentação consistente (4 espaços por nível)
   - Utilize codificação semântica:
     - **Negrito** para conceitos-núcleo
     - _Itálico_ para conceitos dependentes
     - `Código` para termos técnicos específicos
     - Símbolos matemáticos para denotar relações (→, ↔, ⊂, ⊆, ∈)
   - Inclua legenda técnica completa para todos os elementos notacionais

O mapa resultante deve funcionar simultaneamente como instrumento analítico para estruturação do conhecimento e como guia arquitetônico para o desenvolvimento do referencial teórico.
```

### 4.2 Prompt para Análise de Interdependências Conceituais

```markdown
# Prompt: Análise de Interdependências Conceituais

Como Cartógrafo Conceitual especializado em modelagem relacional avançada, execute uma análise sistemática das interdependências entre os seguintes conceitos já mapeados:

[LISTA DE CONCEITOS PRINCIPAIS]

Realize uma análise multidimensional seguindo este protocolo técnico:

1. MATRIZ DE INTERDEPENDÊNCIAS TIPIFICADAS:

   - Construa uma matriz n×n onde conceitos aparecem nas linhas e colunas
   - Para cada interseção (i,j), aplique a seguinte taxonomia relacional precisa:
     - Causal direta (X→Y): Relação onde X produz Y como efeito
     - Causal indireta (X→Z→Y): X influencia Y através de mediador Z
     - Correlacional (X↔Y): Co-ocorrência sem direção causal estabelecida
     - Antagônica (X⊥Y): Contradição ou tensão teórica fundamental
     - Complementar (X⊕Y): Reforço mútuo ou sinergia conceitual
     - Hierárquica inclusiva (X⊃Y): X taxonomicamente contém Y
     - Evolutiva (X⟳Y): X evolui historicamente para Y
   - Quantifique a força relacional em escala 1-5 baseada em:
     - Frequência de menção nas fontes primárias (componente objetivo)
     - Centralidade para a argumentação teórica (componente estrutural)
     - Consenso entre especialistas do campo (componente epistêmico)

2. ANÁLISE DE DINÂMICAS SISTÊMICAS:

   - Para relações causais, construa diagrama de fluxo vectorizado em formato Mermaid
   - Identifique explicitamente:
     - Loops de retroalimentação positiva (amplificação)
     - Loops de retroalimentação negativa (regulação)
     - Pontos de alavancagem sistêmica (onde intervenções produzem efeitos máximos)
     - Cadeias causais críticas (sequências com maior poder explicativo)
   - Calcule a densidade relacional de cada subgrafo (número de conexões/número possível)

3. ANÁLISE DE AGRUPAMENTOS CONCEITUAIS:

   - Aplique algoritmo de clustering baseado na força das conexões
   - Para cada cluster identificado:
     - Derive nome representativo baseado em análise de essência comum
     - Calcule coesão interna (média das conexões intracluster)
     - Identifique conceito central com maior número de conexões internas
   - Mapeie conceitos-ponte que conectam múltiplos clusters
   - Quantifique o papel de intermediação (betweenness centrality) de cada conceito

4. CARTOGRAFIA DE TENSÕES TEÓRICAS:

   - Para cada par antagônico (X⊥Y), produza análise estruturada:
     - Origem histórica da divergência teórica
     - Bases epistemológicas do antagonismo
     - Implicações para o campo de conhecimento
     - Propostas de síntese ou reconciliação teórica na literatura
     - Posições predominantes no debate atual

5. ANÁLISE MÉTRICA DE CENTRALIDADE:
   - Calcule três métricas de centralidade para cada conceito:
     - Centralidade de grau: número total de conexões diretas
     - Centralidade de intermediação: frequência com que o conceito conecta outros
     - Centralidade de autovetor: influência indireta na rede conceitual
   - Identifique os 3-5 conceitos com maior centralidade combinada
   - Justifique tecnicamente por que esses conceitos são estruturalmente fundamentais
   - Recomende ênfase proporcional a cada conceito no referencial teórico

Esta análise estrutural fornecerá as bases para um referencial teórico rigorosamente integrado, com conexões explícitas entre conceitos e clara articulação das tensões e complementaridades teóricas.
```

### 4.3 Prompt para Diagnóstico de Completude Teórica

```markdown
# Prompt: Diagnóstico de Completude Teórica

Como Cartógrafo Conceitual especializado em análise de insuficiências estruturais, examine criticamente o mapa conceitual existente:

[INSERIR MAPA CONCEITUAL ATUAL]

Execute um diagnóstico sistemático seguindo este protocolo analítico:

1. ANÁLISE DE ALINHAMENTO COM OBJETIVOS INVESTIGATIVOS:

   - Compare o mapa atual com os objetivos específicos da pesquisa:
     [LISTAR OBJETIVOS]
   - Para cada objetivo, quantifique o grau de cobertura conceitual em escala numérica (0-100%)
   - Identifique objetivos com cobertura inferior a 70%
   - Especifique com precisão quais dimensões de cada objetivo carecem de representação
   - Proponha adições conceituais específicas vinculadas aos gaps identificados, citando autores-chave

2. AVALIAÇÃO DE PROFUNDIDADE ESTRUTURAL:

   - Realize análise estatística de profundidade atual:
     - Percentual de conceitos com 3+ níveis de profundidade
     - Profundidade média por ramo principal
     - Identificação de ramos com profundidade < 3 níveis
   - Para ramos subdesenvolvidos ("truncados"), forneça:
     - Análise das ramificações ausentes mais críticas
     - Subdivisões conceituais específicas com referências bibliográficas
     - Terminologia técnica precisa para cada nível proposto
   - Priorize expansões com base em relevância para objetivos (alta/média/baixa)

3. ANÁLISE TOPOLÓGICA DE CONECTIVIDADE:

   - Identifique "ilhas conceituais" (subgrafos desconectados) com detecção algorítmica
   - Calcule a distância geodésica média entre conceitos (número médio de passos entre quaisquer dois nós)
   - Identifique pontes lógicas ausentes onde deveria haver conexão direta
   - Para cada insuficiência de conectividade, especifique:
     - Conceitos que deveriam estar conectados
     - Natureza da relação ausente
     - Evidência na literatura que sustenta a conexão
     - Impacto da ausência na integridade estrutural do mapa

4. ANÁLISE DE DIVERSIDADE EPISTÊMICA:

   - Quantifique representação de diferentes perspectivas teóricas:
     - Escola de pensamento dominante no mapa atual (% de conceitos)
     - Escolas sub-representadas ou ausentes
     - Proporção de conceitos de origem empírica vs. teórica
   - Identifique vieses sistemáticos favorecendo determinadas perspectivas
   - Para cada desequilíbrio significativo, recomende:
     - Autores específicos que representam perspectivas alternativas
     - Conceitos-chave ausentes de abordagens subrrepresentadas
     - Obras seminais que deveriam ser incorporadas

5. VALIDAÇÃO EMPÍRICA DOS CONSTRUTOS TEÓRICOS:

   - Classifique cada afirmação causal presente no mapa quanto ao suporte empírico:
     - [SEM EVIDÊNCIA]: Relação teórica sem estudos empíricos citados
     - [EVIDÊNCIA LIMITADA]: Suportada por poucos estudos ou estudos metodologicamente fracos
     - [EVIDÊNCIA MODERADA]: Múltiplos estudos consistentes
     - [EVIDÊNCIA FORTE]: Amplo corpo de pesquisa com resultados convergentes
   - Identifique relações causais críticas sem suporte empírico adequado
   - Recomende estudos específicos que poderiam preencher as lacunas:
     - Tipo de desenho metodológico necessário
     - Variáveis que deveriam ser mensuradas
     - Referências a estudos semelhantes em campos análogos

6. RELATÓRIO TÉCNICO DE INSUFICIÊNCIAS PRIORITÁRIAS:
   - Classifique todas as lacunas usando matriz de criticidade:
     | LACUNA | IMPACTO (1-5) | FACILIDADE DE RESOLUÇÃO (1-5) | PRIORIDADE COMBINADA |
   - Para cada lacuna de alta prioridade (top 30%), forneça:
     - Descrição técnica precisa da insuficiência estrutural
     - Impacto específico na qualidade do referencial teórico
     - Plano de ação detalhado para resolução
     - Designação do agente mais qualificado (Pesquisador/Revisor/etc.)
     - Métricas objetivas para verificar resolução da lacuna

Este diagnóstico produzirá um roteiro técnico para o refinamento sistemático do mapa conceitual, garantindo alinhamento com objetivos, profundidade adequada, coerência estrutural e equilíbrio epistêmico.
```

### 4.4 Prompt para Visualização Cognitiva Adaptativa

```markdown
# Prompt: Visualização Cognitiva Adaptativa

Como Cartógrafo Conceitual especializado em representação multimodal de conhecimento, transforme o mapa conceitual existente em visualizações cognitivamente otimizadas conforme as seguintes especificações:

1. DIAGRAMA TAXONÔMICO HIERÁRQUICO:

   - Implemente representação em árvore utilizando sintaxe Mermaid otimizada para taxonomias
   - Aplique princípios de design informacional:
     - Agrupamento Gestalt para conceitos relacionados
     - Distância intermodal proporcional à distância conceitual
     - Diferenciação cromática para categorias epistemológicas
   - Empregue codificação visual semântica para tipos de conceitos:
     - Retângulos: conceitos teóricos
     - Elipses: construtos metodológicos
     - Hexágonos: evidências empíricas
     - Diamantes: conceitos transversais
   - Implemente sistema de rótulos que maximize a densidade informacional sem comprometer legibilidade

2. GRAFO DE REDE RELACIONAL:

   - Construa representação de rede em formato Mermaid com:
     - Nós diferenciados por tamanho conforme centralidade
     - Arestas direcionadas e não-direcionadas conforme tipo de relação
     - Espessura de linha proporcional à força da relação
     - Clusters visualmente demarcados por proximidade espacial e bordas
   - Aplique algoritmo de layout force-directed para otimização topológica
   - Implemente codificação cromática para visualização instantânea de padrões:
     - Tons quentes (vermelho/laranja): relações causais
     - Tons frios (azul/verde): relações hierárquicas
     - Tons contrastantes (roxo/amarelo): relações antagônicas
   - Adicione mecanismo referencial que vincule elementos visuais às fontes

3. MAPA MENTAL RADIAL BALANCEADO:

   - Desenvolva representação radial com conceito central e ramificações equilibradas
   - Implemente princípios de equilíbrio visual:
     - Distribuição de massa conceitual balanceada entre quadrantes
     - Comprimento de ramos proporcional à importância relativa
     - Densidade informacional uniforme em toda a visualização
   - Utilize formatação avançada em markdown para codificação semântica:
     - **Negrito**: conceitos primários
     - _Itálico_: qualificadores e propriedades
     - `Monospace`: terminologia técnica específica
     - ~~Tachado~~: conceitos contestados ou superados
   - Incorpore microrótulos explicativos em pontos críticos do mapa

4. MATRIZ DE INTERDEPENDÊNCIAS MULTIVARIADA:

   - Construa tabela de duas entradas com:
     - Conceitos principais em ambos os eixos
     - Sistema de notação multissimbólica nas interseções:
       - Símbolos primários para tipo de relação: →, ↔, ⊥, ⊕, ⊃, ⟳
       - Superíndices para força da relação: ¹, ², ³, ⁴, ⁵
       - Subíndices para fonte primária da relação: ₐ, ₑ, ᵢ, ₒ
     - Gradiente cromático para visualização de padrões de cluster
   - Aplique formatação condicional para destacar:
     - Células com relações mais fortes (4-5)
     - Células com relações antagônicas
     - Padrões diagonais ou ortogonais significativos

5. CRONOGRAFIA CONCEITUAL EVOLUTIVA:
   - Desenvolva linha do tempo conceitual com:
     - Eixo temporal preciso com décadas/períodos definidos
     - Posicionamento de conceitos em seu período de emergência
     - Linhas de influência mostrando derivação conceitual
     - Marcos históricos contextuais para o campo
   - Implemente codificação visual para:
     - Períodos de estabilidade vs. ruptura paradigmática
     - Tendências de convergência ou divergência teórica
     - Densidade de desenvolvimento em diferentes períodos
   - Incorpore micro-históricos explicativos para transformações conceituais críticas

Cada visualização deve incluir como elementos auxiliares:

- Metadados técnicos (título, data, versão)
- Sistema de legenda técnica exaustivo
- Notas metodológicas sobre interpretação
- Fontes bibliográficas primárias vinculadas a elementos específicos
- Indicadores de confiança/incerteza para relações representadas

Estas visualizações fornecem múltiplas perspectivas complementares da mesma estrutura conceitual, otimizadas para diferentes operações cognitivas (categorização, relação, sequenciamento) e diferentes perfis de processamento informacional.
```

### 4.5 Prompt para Orquestração de Fluxo Investigativo

```markdown
# Prompt: Orquestração de Fluxo Investigativo

Como Cartógrafo Conceitual especializado em gerenciamento de pesquisa baseado em análise estrutural, examine o mapa conceitual atual e a base de conhecimento para coordenar o trabalho da equipe investigativa. Execute o seguinte protocolo de orquestração:

1. DIAGNÓSTICO DE PRONTIDÃO SECCIONAL:

   - Para cada seção principal do referencial teórico, realize análise multifatorial:
     - Avaliação de Cobertura Conceitual (% dos conceitos necessários presentes)
     - Avaliação de Profundidade Conceitual (níveis médios de desenvolvimento)
     - Avaliação de Suporte Bibliográfico (quantidade e qualidade das referências)
     - Avaliação de Conexões Internas (densidade de conexões entre conceitos da seção)
     - Avaliação de Maturidade Expositiva (clareza das definições e relações)
   - Aplique algoritmo de classificação baseado nos critérios acima:
     - [PRONTO PARA ESCRITA]: Pontuação ≥ 85% em todos os critérios
     - [NECESSITA REVISÃO]: Pontuação 60-84% em um ou mais critérios
     - [NECESSITA PESQUISA]: Pontuação < 60% em um ou mais critérios
   - Gere justificativa técnica detalhada para cada classificação, identificando fatores determinantes

2. PLANEJAMENTO ESTRUTURAL PARA O ESCRITOR:

   - Para seções classificadas como [PRONTO PARA ESCRITA], desenvolva:
     - Arquitetura textual completa com esquema hierárquico de títulos e subtítulos
     - Sequenciamento lógico de conceitos obedecendo princípios de:
       > Progressão do geral para o específico
       > Estabelecimento de fundamentos antes de construtos derivados
       > Agrupamento temático de conceitos relacionados
       > Transições explícitas entre clusters conceituais
     - Mapeamento preciso de fontes a seções específicas, com:
       > Página/parágrafo específico para cada conceito-chave
       > Identificação de citações diretas de alto impacto
       > Contraponto de múltiplas fontes para conceitos centrais
     - Diretrizes retóricas para abertura (contextualização) e fechamento (síntese)
     - Demarcação explícita dos pontos de transição entre subseções

3. DIRECIONAMENTO ANALÍTICO PARA O REVISOR:

   - Para seções classificadas como [NECESSITA REVISÃO], especifique:
     - Lista priorizada de conceitos que requerem clarificação, com:
       > Ambiguidades terminológicas específicas
       > Inconsistências definicionais entre fontes
       > Lacunas explicativas em construtos centrais
     - Mapeamento de relações conceituais subdesenvolvidas:
       > Vínculos causais insuficientemente explicados
       > Hierarquias taxonômicas incompletas
       > Contradições aparentes não resolvidas
     - Protocolo específico de revisão por fonte:
       > Páginas/seções a serem reanalisadas
       > Extração de definições operacionais específicas
       > Identificação de exemplos ilustrativos
     - Questões analíticas precisas formuladas em linguagem técnica
     - Critérios de validação para considerar a revisão satisfatória

4. DIRECIONAMENTO INVESTIGATIVO PARA O PESQUISADOR:

   - Para seções classificadas como [NECESSITA PESQUISA], forneça:
     - Formulação técnica de consultas para busca sistemática:
       > Termos primários e secundários com operadores booleanos
       > Delimitadores temporais, metodológicos e linguísticos
       > Filtros específicos para bases de dados científicas
     - Identificação de autores seminais e obras fundamentais:
       > Pesquisadores com maior índice h no subcampo
       > Obras com maior taxa de citação na última década
       > Centros de pesquisa especializados na temática
     - Taxonomia metodológica de estudos prioritários:
       > Tipo de estudo (metanálise, experimental, survey, etc.)
       > Abordagem metodológica (qualitativa, quantitativa, mista)
       > Delineamento temporal (transversal, longitudinal)
     - Especificação precisa das lacunas informacionais:
       > Formulação de perguntas investigativas em formato PICO
       > Identificação de variáveis-chave para extração
       > Critérios para avaliar qualidade metodológica dos estudos

5. METANÁLISE DE PROGRESSO INVESTIGATIVO:
   - Desenvolva dashboard quantitativo com:
     - Percentual de completude por seção e global
     - Análise de tendência temporal de desenvolvimento
     - Visualização de distribuição de esforço entre agentes
     - Métricas de qualidade (cobertura, profundidade, conexidade)
   - Identifique gargalos críticos através de análise de caminho:
     - Dependências sequenciais entre seções
     - Recursos limitantes (fontes, expertise específica)
     - Complexidade diferencial entre subseções
   - Recomendações de ajuste estratégico baseadas em dados:
     - Realocação de recursos entre áreas temáticas
     - Modificação de escopo (expansão/contração) conforme necessário
     - Estratégias de paralelização para otimizar fluxo de trabalho
   - Estabeleça critérios operacionais de finalização:
     - KPIs objetivos para cada dimensão do mapa conceitual
     - Marcos incrementais claramente definidos
     - Protocolos de validação cruzada entre agentes

Esta orquestração sistemática alinha os esforços de todos os agentes com precisão cirúrgica, otimizando recursos investigativos, eliminando redundâncias e garantindo desenvolvimento equilibrado do referencial teórico.
```

### 4.6 Prompt para Validação Dialógica Estruturada

```markdown
# Prompt: Validação Dialógica Estruturada

Como Cartógrafo Conceitual especializado em comunicação estruturada e validação epistêmica, prepare um relatório técnico do mapa conceitual atual para validação com o Orientador. Construa o seguinte pacote comunicacional:

1. SÍNTESE ESTRUTURAL EXECUTIVA:

   - Construa um parágrafo tecnicamente denso que:
     - Articule a macroestrutura do mapa em linguagem sistêmica
     - Quantifique a complexidade estrutural (nós, arestas, profundidade, clusters)
     - Destaque os princípios organizacionais que guiaram a construção
     - Identifique as relações estruturalmente mais significativas
   - Utilize linguagem precisa que:
     - Evite ambiguidades terminológicas
     - Reflita o estado atual de desenvolvimento
     - Comunique tanto conquistas quanto limitações atuais
     - Sugira o potencial investigativo do mapa em sua forma atual

2. VISUALIZAÇÃO ESTRATÉGICA SIMPLIFICADA:

   - Desenvolva representação gráfica minimalista que:
     - Preserve apenas conceitos de primeiro e segundo nível
     - Mantenha todas as relações entre conceitos principais
     - Utilize codificação visual simplificada mas informacionalmente rica
     - Equilibre complexidade representacional com clareza interpretativa
   - Implemente sistema de numeração hierárquica que:
     - Facilite referenciação precisa durante discussão
     - Reflita a estrutura taxonômica subjacente
     - Permita mapeamento direto com documento textual futuro
     - Suporte navegação eficiente durante revisão

3. MATRIZ DE ALINHAMENTO OBJETIVO-CONCEITUAL:

   - Para cada objetivo específico da pesquisa, construa análise estruturada:
     - Mapeamento completo de conceitos primários e secundários vinculados
     - Representação visual da rede conceitual específica para o objetivo
     - Quantificação de cobertura atual (% do necessário para abordar objetivo)
     - Análise de suficiência explicativa (capacidade de responder questões-chave)
   - Para cada objetivo, identifique:
     - Núcleo conceitual essencial (conceitos sem os quais o objetivo não pode ser atingido)
     - Conceitos periféricos (enriquecem mas não são essenciais)
     - Lacunas conceptuais críticas (ausências que comprometem o objetivo)
     - Oportunidades de integração com outros objetivos (sobreposições estratégicas)

4. PONTOS DE DECISÃO ESTRATÉGICA:

   - Formule 3-5 questões cruciais que exigem input epistêmico do Orientador:
     - Dilemas classificatórios onde múltiplas taxonomias são defensáveis
     - Demarcações de escopo onde inclusão/exclusão temática é ambígua
     - Balanceamento entre perspectivas teóricas concorrentes
     - Priorização entre profundidade vertical vs. abrangência horizontal
   - Para cada ponto decisório, desenvolva:
     - Formulação precisa do dilema em termos técnicos
     - Análise das implicações estruturais de cada alternativa
     - Mapeamento de precedentes na literatura científica
     - Matriz de prós e contras baseada em critérios explicitos

5. ROTEIRO ESTRATÉGICO DE DESENVOLVIMENTO:
   - Com base no estado atual, projete:
     - Sequência ótima de expansões conceituais prioritárias
     - Estimativa de esforço investigativo para cada fase (em horas-pesquisa)
     - Pontos de controle intermediários com entregáveis bem definidos
     - Dependências críticas entre diferentes linhas de desenvolvimento
   - Estabeleça critérios técnicos para "finalização suficiente":
     - Métricas quantitativas de completude estrutural
     - Indicadores qualitativos de maturidade conceitual
     - Protocolos de validação cruzada entre diferentes visualizações
     - Benchmarks comparativos com referenciais teóricos análogos
```
