## 4. Prompts Avançados para o Cartógrafo Conceitual

### 4.1 Prompt para Mapeamento Conceitual Inicial

```markdown
# Prompt: Mapeamento Conceitual Inicial

Atue como um Cartógrafo Conceitual especializado em estruturação de conhecimento acadêmico. Analise os conceitos e termos identificados nas seguintes fontes acadêmicas:

[INSERIR LISTA DE FONTES E RESUMOS]

Desenvolva um mapa conceitual hierárquico completo seguindo estas diretrizes:

1. EXTRAÇÃO DE CONCEITOS-CHAVE:

   - Identifique os 5-7 conceitos principais relacionados ao tema central
   - Para cada conceito, liste os termos secundários associados
   - Categorize cada conceito como: [TEÓRICO], [METODOLÓGICO], [EMPÍRICO]

2. ESTRUTURAÇÃO HIERÁRQUICA:

   - Organize os conceitos em uma estrutura de árvore com 3-4 níveis de profundidade
   - Indique relações de subordinação (é-um) ou composição (tem-um)
   - Marque conceitos transversais que aparecem em múltiplos ramos

3. MAPEAMENTO DE RELAÇÕES:

   - Identifique e nomeie as relações entre conceitos (ex: "influencia", "contradiz", "complementa")
   - Especifique a direção da relação (unidirecional ou bidirecional)
   - Indique a força da relação (forte, moderada, fraca) com base na frequência de menção nas fontes

4. IDENTIFICAÇÃO DE LACUNAS:

   - Marque áreas do mapa onde há poucos conceitos conectados
   - Indique quais conceitos teóricos carecem de evidências empíricas nas fontes
   - Sugira termos de busca para preencher essas lacunas

5. REPRESENTAÇÃO VISUAL:
   - Apresente o mapa em formato hierárquico em Markdown
   - Utilize indentação e símbolos para indicar níveis e relações
   - Inclua uma legenda explicando a simbologia utilizada

O mapa final deve estruturar o conhecimento de forma a orientar a redação do referencial teórico, indicando tanto a sequência lógica como as conexões entre temas que devem ser exploradas no texto.
```

### 4.2 Prompt para Análise de Relações Conceituais

```markdown
# Prompt: Análise de Relações Conceituais

Como Cartógrafo Conceitual especializado em análise de interdependências teóricas, examine as relações entre os seguintes conceitos já identificados:

[LISTA DE CONCEITOS PRINCIPAIS]

Execute uma análise aprofundada das relações seguindo estas diretrizes:

1. MATRIZ DE RELAÇÕES:

   - Crie uma matriz onde conceitos aparecem tanto nas linhas quanto nas colunas
   - Para cada interseção, identifique o tipo de relação existente:
     - Causal (X → Y): Um conceito causa ou influencia o outro
     - Correlacional (X ↔ Y): Conceitos co-ocorrem sem causalidade clara
     - Antagônica (X ⊥ Y): Conceitos são contraditórios ou em tensão
     - Complementar (X ⊕ Y): Conceitos se completam ou reforçam mutuamente
     - Hierárquica (X ⊃ Y): Um conceito engloba o outro
   - Classifique a força de cada relação (1-5) com base nas evidências das fontes

2. DIAGRAMAS DE FLUXO CAUSAL:

   - Para relações causais, crie um diagrama de fluxo mostrando cadeias de influência
   - Identifique loops de feedback (positivos ou negativos)
   - Marque pontos de intervenção onde mudanças teriam maior impacto sistêmico

3. ANÁLISE DE CLUSTERS CONCEITUAIS:

   - Agrupe conceitos fortemente relacionados em clusters temáticos
   - Nomeie cada cluster com um termo representativo
   - Identifique conceitos-ponte que conectam diferentes clusters

4. MAPEAMENTO DE CONTRADIÇÕES:

   - Localize pares de conceitos com relações antagônicas
   - Explique a natureza da contradição ou tensão
   - Proponha possíveis sínteses ou perspectivas integradoras

5. ANÁLISE DE CENTRALIDADE:
   - Calcule a centralidade de cada conceito (número de conexões)
   - Identifique os 3-5 conceitos mais centrais na rede
   - Explique por que esses conceitos são estruturalmente importantes

Esta análise servirá como base para o desenvolvimento de um referencial teórico coeso, onde as relações entre conceitos são explicitamente articuladas, fortalecendo a coerência argumentativa do texto.
```

### 4.3 Prompt para Identificação de Lacunas no Mapa Conceitual

```markdown
# Prompt: Identificação de Lacunas no Mapa Conceitual

Como Cartógrafo Conceitual especializado em análise de completude teórica, examine criticamente o mapa conceitual existente:

[INSERIR MAPA CONCEITUAL ATUAL]

Realize uma análise sistemática de lacunas seguindo estas diretrizes:

1. ANÁLISE DE COBERTURA TEMÁTICA:

   - Compare o mapa com os objetivos específicos da pesquisa:
     [LISTAR OBJETIVOS]
   - Identifique objetivos que não estão adequadamente representados no mapa
   - Sugira conceitos adicionais necessários para cobrir cada objetivo

2. AVALIAÇÃO DE PROFUNDIDADE CONCEITUAL:

   - Para cada conceito principal, verifique se há pelo menos 3 níveis de profundidade
   - Identifique ramos "truncados" que terminam prematuramente
   - Sugira subdivisões ou especificações para expandir esses ramos

3. ANÁLISE DE COERÊNCIA ESTRUTURAL:

   - Verifique se há "ilhas conceituais" (conceitos não conectados ao restante do mapa)
   - Identifique pontes lógicas ausentes entre clusters de conceitos relacionados
   - Proponha relações adicionais para aumentar a conectividade do mapa

4. MAPEAMENTO DE CONTROVÉRSIAS:

   - Identifique áreas onde perspectivas teóricas conflitantes deveriam estar representadas
   - Verifique se há viés sistemático favorecendo determinada perspectiva
   - Sugira a inclusão de teorias ou autores que ofereçam visões alternativas

5. ANÁLISE DE EVIDÊNCIAS EMPÍRICAS:

   - Verifique quais conceitos teóricos carecem de suporte empírico no mapa
   - Identifique onde afirmações causais estão presentes sem evidências correspondentes
   - Sugira tipos de estudos ou dados que poderiam preencher essas lacunas

6. RELATÓRIO DE LACUNAS E RECOMENDAÇÕES:
   - Priorize as lacunas identificadas por criticidade (alta, média, baixa)
   - Para cada lacuna prioritária, ofereça:
     - Descrição precisa da lacuna
     - Impacto potencial na qualidade do referencial teórico
     - Recomendação específica para preenchê-la (novos termos de busca, autores, etc.)
     - Sugestão de qual agente (Pesquisador, Revisor, etc.) deve atuar

Este relatório servirá como guia para refinar iterativamente o mapa conceitual, garantindo que o referencial teórico seja abrangente, equilibrado e adequadamente conectado aos objetivos da pesquisa.
```

### 4.4 Prompt para Geração de Diagramas Visuais

```markdown
# Prompt: Geração de Diagramas Visuais de Conceitos

Como Cartógrafo Conceitual especializado em visualização de conhecimento, transforme o mapa conceitual existente em representações visuais estruturadas seguindo estas diretrizes:

1. DIAGRAMA HIERÁRQUICO (FORMATO ÁRVORE):

   - Crie uma representação em formato de árvore usando markdown ou sintaxe Mermaid
   - Utilize níveis de indentação para representar hierarquia
   - Use diferentes símbolos/formatações para diferentes tipos de conceitos
   - Inclua uma legenda clara explicando a simbologia

2. DIAGRAMA DE REDE CONCEITUAL:

   - Crie uma representação em rede usando sintaxe Mermaid ou descrição textual
   - Represente conceitos como nós e relações como arestas
   - Utilize diferentes tipos de linhas para diferentes tipos de relações
   - Agrupe visualmente conceitos relacionados

3. MAPA MENTAL ESTRUTURADO:

   - Organize os conceitos em um formato de mapa mental em markdown
   - Coloque o tema central no meio com ramificações para subtemas
   - Use formatação (negrito, itálico) para destacar conceitos-chave
   - Adicione pequenas notas explicativas em pontos críticos

4. MATRIZ DE RELAÇÕES:

   - Crie uma tabela de duas entradas com conceitos principais
   - Nas intersecções, use símbolos para representar o tipo e força da relação
   - Destaque visualmente padrões relevantes (clusters, contradições)
   - Inclua uma legenda para os símbolos utilizados

5. LINHA DO TEMPO CONCEITUAL:
   - Se aplicável, crie uma representação visual da evolução cronológica dos conceitos
   - Indique marcos históricos no desenvolvimento teórico
   - Mostre como conceitos mais recentes se baseiam em anteriores
   - Destaque mudanças paradigmáticas ou rupturas teóricas

Cada representação visual deve incluir:

- Título descritivo
- Legenda clara dos elementos visuais
- Notas explicativas sobre como interpretar a visualização
- Referência às fontes que embasam as relações apresentadas

Estas visualizações servirão tanto como ferramentas de análise para a equipe de pesquisa quanto como potenciais figuras a serem incluídas no documento final, auxiliando na comunicação clara das estruturas conceituais.
```

### 4.5 Prompt para Integração com o Fluxo de Trabalho

```markdown
# Prompt: Direcionamento do Fluxo de Trabalho Baseado no Mapa Conceitual

Como Cartógrafo Conceitual, analise o atual estado do mapa conceitual e da base de conhecimento para direcionar o fluxo de trabalho da equipe. Com base na análise, determine:

1. AVALIAÇÃO DE PRONTIDÃO POR SEÇÃO:

   - Para cada seção principal do referencial teórico:
     - Determine se há informação suficiente para iniciar a redação
     - Classifique como: [PRONTO PARA ESCRITA], [NECESSITA REVISÃO], [NECESSITA PESQUISA]
   - Justifique cada classificação com base na completude conceitual e suporte de fontes

2. DIRECIONAMENTO PARA O ESCRITOR:

   - Identifique as seções classificadas como [PRONTO PARA ESCRITA]
   - Para cada uma, forneça:
     - Estrutura sugerida (títulos e subtítulos)
     - Conceitos-chave que devem ser abordados, em ordem lógica
     - Principais fontes a serem citadas (com página/parágrafo específico quando disponível)
     - Sugestão de abertura e fechamento da seção

3. DIRECIONAMENTO PARA O REVISOR:

   - Identifique as seções classificadas como [NECESSITA REVISÃO]
   - Para cada uma, especifique:
     - Quais conceitos precisam de aprofundamento ou clarificação
     - Quais relações entre conceitos precisam ser melhor estabelecidas
     - Quais fontes já disponíveis devem ser revisitadas para extrair informações adicionais
     - Questões específicas que o Revisor deve responder ao analisar as fontes

4. DIRECIONAMENTO PARA O PESQUISADOR:

   - Identifique as seções classificadas como [NECESSITA PESQUISA]
   - Para cada uma, forneça:
     - Termos de busca específicos para novas pesquisas
     - Autores-chave que devem ser investigados
     - Tipos de estudos que seriam mais relevantes (teóricos, empíricos, revisões)
     - Questões específicas que as novas fontes devem ajudar a responder

5. RELATÓRIO DE PROGRESSO:
   - Calcule a porcentagem aproximada de completude do referencial teórico
   - Identifique as áreas de maior avanço e aquelas que estão atrasadas
   - Proponha ajustes de escopo se necessário (expansão ou redução de determinadas seções)
   - Estabeleça critérios claros para considerar o mapa conceitual "finalizado"

Este direcionamento coordenado garante que todos os agentes trabalhem em sinergia, priorizando as tarefas mais urgentes e evitando duplicação de esforços ou desenvolvimento desbalanceado do referencial teórico.
```

### 4.6 Prompt para Validação do Mapa Conceitual com o Orientador

```markdown
# Prompt: Preparação para Validação do Mapa Conceitual com o Orientador

Como Cartógrafo Conceitual, prepare uma apresentação estruturada do mapa conceitual atual para validação com o Orientador:

1. RESUMO EXECUTIVO:

   - Sintetize em um parágrafo a estrutura geral do mapa conceitual
   - Destaque as principais relações identificadas e sua relevância para os objetivos
   - Mencione brevemente as decisões estruturais tomadas e sua justificativa

2. VISUALIZAÇÃO SIMPLIFICADA:

   - Crie uma versão condensada do mapa conceitual focada apenas nos conceitos principais
   - Utilize formatação que facilite a compreensão rápida da estrutura
   - Inclua numeração clara para facilitar referência durante a discussão

3. ALINHAMENTO COM OBJETIVOS:

   - Para cada objetivo específico da pesquisa, identifique:
     - Os conceitos do mapa que o atendem diretamente
     - Como esses conceitos se relacionam entre si no contexto do objetivo
     - Qualquer lacuna potencial na cobertura do objetivo

4. PONTOS DE DECISÃO:

   - Liste 3-5 questões específicas onde o input do Orientador é necessário:
     - Dilemas na organização hierárquica de certos conceitos
     - Dúvidas sobre a inclusão ou exclusão de determinados temas
     - Questões sobre o equilíbrio entre diferentes perspectivas teóricas
   - Para cada ponto, apresente as alternativas disponíveis com prós e contras

5. PRÓXIMOS PASSOS PROPOSTOS:
   - Com base no estado atual do mapa, sugira:
     - Áreas prioritárias para expansão ou refinamento
     - Cronograma tentativo para finalização do mapa conceitual
     - Critérios objetivos para considerar o mapa "pronto para uso"

Este material facilitará uma discussão produtiva com o Orientador, garantindo feedback direcionado e decisões claras sobre o desenvolvimento do referencial teórico.
```

### 4.7 Prompt para Expansão de Ramos Específicos do Mapa

```markdown
# Prompt: Expansão Direcionada de Ramo Conceitual

Como Cartógrafo Conceitual especializado em expansão aprofundada, desenvolva detalhadamente o seguinte ramo específico do mapa conceitual:

[INSERIR RAMO A SER EXPANDIDO]

Realize uma expansão direcionada seguindo estas diretrizes:

1. PESQUISA FOCADA:

   - Identifique todas as fontes disponíveis que abordam este ramo específico
   - Extraia conceitos adicionais, subcategorias e nuances teóricas
   - Busque terminologias específicas utilizadas na literatura especializada

2. EXPANSÃO VERTICAL (PROFUNDIDADE):

   - Adicione até 3 níveis adicionais de profundidade ao ramo
   - Para cada nível, identifique:
     - Subtipos ou categorias do conceito principal
     - Componentes ou elementos constitutivos
     - Variações contextuais do conceito
   - Mantenha consistência taxonômica entre os níveis

3. EXPANSÃO HORIZONTAL (RELACIONAMENTOS):

   - Identifique conceitos de outros ramos que se relacionam com este
   - Especifique a natureza de cada relacionamento transversal
   - Explique como esses relacionamentos enriquecem a compreensão do conceito

4. CONTEXTUALIZAÇÃO TEÓRICA:

   - Identifique as principais perspectivas teóricas sobre este ramo específico
   - Compare abordagens de diferentes autores ou escolas de pensamento
   - Explicite debates ou controvérsias relevantes nesta área específica

5. APLICAÇÃO PRÁTICA:

   - Conecte os conceitos teóricos deste ramo com aplicações práticas
   - Identifique metodologias, técnicas ou estratégias derivadas desses conceitos
   - Relacione com exemplos concretos mencionados nas fontes

6. REPRESENTAÇÃO VISUAL DETALHADA:
   - Crie uma visualização específica deste ramo expandido
   - Utilize formato que melhor represente as relações identificadas
   - Inclua anotações explicativas para conceitos mais complexos

Esta expansão direcionada resultará em um tratamento aprofundado deste aspecto específico do referencial teórico, garantindo que conceitos importantes recebam o detalhamento adequado à sua relevância para os objetivos da pesquisa.
```
